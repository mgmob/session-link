# Roadmap

Backlog кандидатов на следующие функции пакета (порядок — не приоритет).

- **История handoff'ов** — навигация по цепочке прошлых передач контекста
  (обзор + переход к любой более ранней сессии без ручного поиска файлов).
- **Мульти-драйверное авторство** — создание handoff'ов со стороны других агентов
  (например, Claude Code, Qwen), а не только чтение/опрос из pi.
- **Веб-UI статуса** — наблюдение за текущим handoff и связанной сессией
  (кто кому ответил, есть ли ожидающие уточнения) в браузере.
